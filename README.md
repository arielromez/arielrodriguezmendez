# Ariel Rodriguez Mendez — Portfolio

Static portfolio site built with HTML, CSS, and JavaScript.

## Run locally (macOS)

1. From project root (`.`), start a local server:
   `python3 -m http.server 8000`
2. Open:
   `http://localhost:8000`

## Notes

- Pages are under `pages/`.
- Shared styles/scripts are in `styles.css` and `script.js`.
- Accessibility updates include discernible labels for icon-only navigation buttons.
