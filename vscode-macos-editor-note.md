# VS Code macOS Quick View Note

In VS Code on macOS, the fastest way is:

- Turn on word wrap: press Option+Z
- Fit more content on screen:
  - Hide sidebar: Command+B
  - Hide panel: Command+J
  - Use Zen mode: Command+K then Z
- Jump through the full file quickly:
  - Go to top: Command+Up
  - Go to bottom: Command+Down
- See file structure: Command+Shift+O (if available for that file type)
